# escook-theme

彬哥魔改的 typora 主题。

## 安装

-  首先确定已安装 [Typora](https://typora.io/)
-  通过`设置 -> 外观 -> 打开主题文件夹`打开 theme 目录
-  复制解压后目录内的所有文件到 `theme` 目录下然后重启, 选择菜单栏 -> 主题即可
